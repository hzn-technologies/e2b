##### write()

> **write**(`path`, `data`, `opts`?): `Promise`\<`EntryInfo`\>

Writes content to a file on the path.
  When writing to a file that doesn't exist, the file will get created.
  When writing to a file that already exists, the file will get overwritten.
  When writing to a file that's in a directory that doesn't exist, the directory will get created.

###### Parameters

• **path**: `string`

Path to a new file. For example '/dirA/dirB/newFile.txt' when creating 'newFile.txt'

• **data**: `string` \| `ArrayBuffer` \| `Blob` \| `ReadableStream`\<`any`\>

Data to write to a new file

• **opts?**: `FilesystemRequestOpts`

Options for the request

###### Returns

`Promise`\<`EntryInfo`\>

Information about the written file
