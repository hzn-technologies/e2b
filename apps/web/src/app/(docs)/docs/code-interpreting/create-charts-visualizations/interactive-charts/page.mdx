# Interactive charts

E2B also allows you to create interactive charts with custom styling.

E2B automatically detects charts when executing Python code with `runCode()` in JavaScript or `run_code()` in Python. The Python code must include Matplotlib charts.

When a chart is deteted, E2B sends the data of the chart back to the client. You can access the chart in the `execution.results` array where each item is a `Result` object with the `graph` property.

Here's a simple example of bar chart:
<CodeGroup>
```js
import { Sandbox, BarGraph } from '@e2b/code-interpreter'

const code = `
import matplotlib.pyplot as plt

# Prepare data
authors = ['Author A', 'Author B', 'Author C', 'Author D']
sales = [100, 200, 300, 400]

# Create and customize the bar graph
plt.figure(figsize=(10, 6))
plt.bar(authors, sales, label='Books Sold', color='blue')
plt.xlabel('Authors')
plt.ylabel('Number of Books Sold')
plt.title('Book Sales by Authors')

# Display the graph
plt.tight_layout()
plt.show()
`

const sandbox = await Sandbox.create()
const result = await sandbox.runCode(code)
const graph = result.results[0].graph as BarGraph

console.log('Type:', graph.type)
console.log('Title:', graph.title)
console.log('X Label:', graph.x_label)
console.log('Y Label:', graph.y_label)
console.log('X Unit:', graph.x_unit)
console.log('Y Unit:', graph.y_unit)
console.log('Elements:', graph.elements)
```
```python
from e2b_code_interpreter import Sandbox

code = """
import matplotlib.pyplot as plt

# Prepare data
authors = ['Author A', 'Author B', 'Author C', 'Author D']
sales = [100, 200, 300, 400]

# Create and customize the bar graph
plt.figure(figsize=(10, 6))
plt.bar(authors, sales, label='Books Sold', color='blue')
plt.xlabel('Authors')
plt.ylabel('Number of Books Sold')
plt.title('Book Sales by Authors')

# Display the graph
plt.tight_layout()
plt.show()
"""

sandbox = Sandbox()
execution = sandbox.run_code(code)
graph = execution.results[0].graph

print('Type:', graph.type)
print('Title:', graph.title)
print('X Label:', graph.x_label)
print('Y Label:', graph.y_label)
print('X Unit:', graph.x_unit)
print('Y Unit:', graph.y_unit)
print('Elements:')
for element in graph.elements:
    print('\n  Label:', element.label)
    print('  Value:', element.value)
    print('  Group:', element.group)
```
</CodeGroup>
