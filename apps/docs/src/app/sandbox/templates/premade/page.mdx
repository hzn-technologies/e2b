### Features
Sandbox with Python 3 and data science Python libraries like `pandas`, `numpy`, or `matplotlib`.
- See the complete list of pre-installed Python packages [here](https://github.com/e2b-dev/e2b/blob/main/templates/data_analysis/requirements.txt).
- Ideal for the Data Analysis and Code Interpreter use cases.


### Usage
You can use all the regular [Sandbox API](/api/envs) inside the Code Interpreter / Data Analysis sandbox. On top of that, you can use a few extra methods specifcially made for the use case of code interpreter powered by an LLM.

- **Create sandbox**
<CodeGroup isRunnable={false}>
```js {{ language: 'js' }}
import { DataAnalysis } from '@e2b/sdk'
const sandbox = await DataAnalysis.create()

// Close sandbox once you're done
await sandbox.close()
```

```python {{ language: 'python' }}
from e2b import DataAnalysis
sandbox = DataAnalysis()

# Close sandbox once you're done
sandbox.close()
```
</CodeGroup>

- **Run Python code**
<CodeGroup isRunnable={false}>
```js {{ language: 'js' }}
import { DataAnalysis } from '@e2b/sdk'

// 1. Create sandbox
const sandbox = await DataAnalysis.create()

// 2. Get code that was generated by LLM and run it
const llmGeneratedPythonCode = 'print("Hello World")'
// `artifacts` are chart files created by matplotlib
const { stdout, stderr, artifacts } = await sandbox.runPython(llmGeneratedPythonCode)

// 3. Retrieve chart files:
for (const artifact of artifacts) {
  // Now you can save this file, send it to frontend, or anything else
  const downloadedChart = await artifact.download()
}

// 4. Close sandbox
await sandbox.close
```

```python {{ language: 'python' }}
from e2b import DataAnalysis

# 1. Create sandbox
sandbox = DataAnalysis()

# 2. Get code that was generated by LLM and run it
llm_generated_python_code = 'print("Hello World")'
# `artifacts` are chart files created by matplotlib
stdout, stderr, artifacts = sandbox.run_python(llm_generated_python_code)

# 3. Retrieve chart files:
for artifact in artifacts:
  # Now you can save this file, send it to frontend, or anything else
  downloaded_chart = artifact.download()

# 4. Close sandbox
sandbox.close()
```
</CodeGroup>

- **Install Python packages**

Installs Python packages using `pip`.
<CodeGroup isRunnable={false}>
```js {{ language: 'js' }}
import { DataAnalysis } from '@e2b/sdk'

// 1. Create sandbox
const sandbox = await DataAnalysis.create()

// 2. Install Python package
await DataAnalysis.installPythonPackages('ffmpeg')

// 3. Close sandbox
await sandbox.close()
```
```python {{ language: 'python' }}
from e2b import DataAnalysis

# 1. Create sandbox
sandbox = DataAnalysis()

# 2. Install Python package
sandbox.install_python_packages('ffmpeg')

# 3. Close sandbox
sandbox.close()
```
</CodeGroup>

- **Install system packages**

Installs system packages using `apt-get`.
<CodeGroup isRunnable={false}>
```js {{ language: 'js' }}
import { DataAnalysis } from '@e2b/sdk'

// 1. Create sandbox
const sandbox = await DataAnalysis.create()

// 2. Install system package
await DataAnalysis.installSystemPackages('ffmpeg')

// 3. Close sandbox
await sandbox.close()
```
```python {{ language: 'python' }}
from e2b import DataAnalysis

# 1. Create sandbox
sandbox = DataAnalysis()

# 2. Install system package
sandbox.install_system_packages('ffmpeg')

# 3. Close sandbox
sandbox.close()
```
</CodeGroup>
