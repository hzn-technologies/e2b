# Premade Sandboxes

To get your started quickly, we offer a two premade sandboxes.

## `Base`
A "base" sandbox with Nodejs, Python 3, and essentials libraries like `curl`.
- Ideal for getting started with the SDK.
- Inspect the template code [here](https://github.com/e2b-dev/e2b/blob/main/templates/base/Dockerfile).

## `DataAnalysis`
Sandbox with Python 3 and data science Python libraries like `pandas`, `numpy`, `matplotlib`.
- Ideal for the Data Analysis and Code Interpreter use cases.
- Inspect the template code [here](https://github.com/e2b-dev/e2b/blob/main/templates/base/Dockerfile).

<CodeGroup title="Create DataAnalysis sandbox" isRunnable={false}>
```js {{ language: 'js' }}
import { Sandbox } from '@e2b/sdk'

const sandbox = await Sandbox.create({
  id: 'Python3-DataAnalysis',
})

await sandbox.close()
```

```python {{ language: 'python' }}
from e2b import Sandbox

sandbox = Sandbox(
  id="Python3-DataAnalysis",
)

sandbox.close()
```
</CodeGroup>
