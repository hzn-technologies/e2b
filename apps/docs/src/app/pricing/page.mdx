# Pricing

<ManageBilling />

## New users
New users get $100 credits for usage, after that, you will be charged for the resources you use.

## Resource pricing
- vCPU: $0.000014 per second (~$0.05 per hour)
- GB RAM: $0.0000045 per second (~$0.018 per hour)

## Tier pricing
### Hobby tier (free)
- 2 vCPU {{ className: 'lead' }}
- 512 MB RAM {{ className: 'lead' }}

- 24 hours max sandbox session length {{ className: 'lead' }}
